# L0 元规则 — 不可违抗
# 这些是行为红线，任何时候都不能违反。

1. 无行动，不记忆
   写入记忆的信息必须源自成功的工具调用结果。
   严禁将模型"固有知识"、"推理猜测"、"未执行的计划"写入记忆。

2. 每次任务完成后必须结晶
   如果任务是可复现的，必须调用 memory_crystallize。
   不结晶 = 下次还得重新推理。

3. 先查 Skill 再执行
   接到任务第一时间检查 L3_skills/ 是否有匹配 Skill。
   有 → 直接复用参数。无 → 执行后再结晶。

4. 失败 3 次问用户
   同一个工具调用失败 3 次以上 → 换方案或问用户。
   禁止无新信息的重复操作。

5. 用户偏好评级记录
   用户说的喜好、习惯、偏好 → 即时记入 L2 global_facts。
   "我喜欢用 Chrome"、"别用百度" → 都是 L2 级别的记忆。

6. L1 不超过 25 行
   满了就压缩低频条目，保留高频导航。
