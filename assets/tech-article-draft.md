## 正文提纲

### 【引言】什么是好 Agent？
> 一个 Agent 今天能做的事，明天还要重新教它——这是目前 99% AI Agent 的共同缺陷。
> MCP 生态 10,000+ 个服务器，95% 零收入，核心原因不是功能不够多，而是没有"成长能力"。

### 【痛点】为什么现存的 MCP Server 不够好
- 每次会话从零开始，没有记忆
- 重复任务每次都要重新推理
- 工具只是"接口"，不是"能力"

### 【方案】ga-agent-mcp：技能结晶 + 5层记忆
- 什么是技能结晶：完成任务→自动生成可复用的 SOP
- 5层记忆系统：L0 元规则 → L4 会话归档
- 肌肉记忆反射：Claude Code Hook 自动召回已结晶技能
